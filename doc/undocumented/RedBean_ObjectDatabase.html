		load (line 302)
	
 
	
	

This interface describes how ANY Object Database should  behave.For detailed descriptions of RedBean specific implementation  see: RedBean_OODB.


An Object Database should be able to load a bean using a $type and $id.  The $type argument indicated what kind of bean you are looking for.  The $id argument specifies the primary key ID; which links the bean to  a (series) of record(s) in the database.
