		Constructor __construct (line 4237)
	
 
	
	

Constructor.


The Constructor of the ToolBox takes three arguments: a RedBean_OODB $redbean  object database, a RedBean_Adapter $databaseAdapter and a  RedBean_QueryWriter $writer. It stores these objects inside and acts as  a micro service locator. You can pass the toolbox to any object that needs  one of the RedBean core objects to interact with.
