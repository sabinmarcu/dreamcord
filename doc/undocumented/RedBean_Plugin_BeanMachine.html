Adds a new Group to the Query.


Usage:
$q->addGroup("WHERE-CLAUSE", " WHERE @ ", " AND ");  $q->add(" color = :color ");  $q->add(" smell = :smell ");  (Outputs: WHERE color = :color AND smell = :smell )
