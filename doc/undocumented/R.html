		static associate (line 7604)
	
 
	
	

Associates two Beans. This method will associate two beans with eachother.


You can then get one of the beans by using the related() function and  providing the other bean. You can also provide a base bean in the extra  parameter. This base bean allows you to add extra information to the association  record. Note that this is for advanced use only and the information will not  be added to one of the beans, just to the association record.  It's also possible to provide an array or JSON string as base bean. If you  pass a scalar this function will interpret the base bean as having one  property called 'extra' with the value of the scalar.
