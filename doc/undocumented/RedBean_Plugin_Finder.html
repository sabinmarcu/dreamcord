		static where (line 5722)
	
 
	
	

Fetches a collection of OODB Bean objects based on the SQL  criteria provided. For instance;


  • RedBean_Plugin_Finder::where("page", " name LIKE '%more%' ");
  Will return all pages that have the word 'more' in their name.  The second argument is actually just plain SQL; the function expects  this SQL to be compatible with a SELECT * FROM TABLE WHERE X query,  where X is ths search string you provide in the second parameter.  Another example, using slots:
  • RedBean_Plugin_Finder::where("page", " name LIKE :str ",array(":str"=>'%more%'));
  Also, note that the default search is always 1. So if you do not  specify a search parameter this function will just return every  bean of the given type:
  • RedBean_Plugin_Finder::where("page");
