		optimize (line 6768)
	
 
	
	

Called by the optimizer. This asks the plugin to optimize  the table based on column and value information provided.


If the optimize() method returns false, no further optimizations  are allowed. In case of true the optimizer will advance to the next  optimizer in the collection.
