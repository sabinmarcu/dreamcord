		static canBeTreatedAsInt (line 2353)
	
 
	
	

This is a utility service method publicly available.


It allows you to check whether you can safely treat an certain value as an integer by  comparing an int-valled string representation with a default string casted string representation and  a ctype-digit check. It does not take into account numerical limitations (X-bit INT), just that it  can be treated like an INT. This is useful for binding parameters to query statements like  Query Writers and drivers can do.

	
    
				
  • return: boolean result of analysis
